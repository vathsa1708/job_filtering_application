# Job Search Application

## Description

This is a web application for searching job listings. Users can filter job listings based on various criteria such as minimum experience, company name, location, remote or on-site jobs, tech stack, role, and minimum base pay.

## Features

- Search for job listings based on different criteria.
- Filter job listings by minimum experience, company name, location, remote or on-site jobs, tech stack, role, and minimum base pay.
- Responsive design for mobile and desktop use.
- Infinite scroll for loading more job listings as the user scrolls down.

## Technologies Used

- React.js
- Redux
- CSS (for styling)

## Installation

1. **Clone the repository:**

```bash
git clone <repository-url>
```
2. **Install dependencies:**

```bash
npm install
```

3. **Start the development server:**

```bash
npm start
```

4. **Open your browser and visit [http://localhost:3000](http://localhost:3000) to view the application.**

## Usage

- Enter search criteria in the input fields to filter job listings.
- Scroll down to load more job listings.
